# flater

A Flutter starter app with a custom welcome screen.

## Getting Started

This project was scaffolded locally with Flutter and already includes a responsive welcome screen.

Run the app from this folder with the local SDK:

```powershell
.\tools\flutter\bin\flutter.bat pub get
.\tools\flutter\bin\flutter.bat run
```

Useful commands:

```powershell
.\tools\flutter\bin\flutter.bat test
.\tools\flutter\bin\flutter.bat doctor
```
